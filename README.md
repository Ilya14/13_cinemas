# 13_cinemas

## Описание

Скрипт выводит в отсортированном по рейтингу виде самые популярные фильмы, 
идущие в данный момент в кинотеатрах.

## Использование

Скрипт принимает следующие необязательные параметры:

* movies_count - количество фильмов для вывода в консоль (по умолчанию 10);

* cinemas_count_limit - нижняя граница для количества кинотеатров,
в которых идет данный фильм (по умолчанию 1).

Для получения справки можно использовать аргументы -h или --help:

```sh
$ python3.5 ./cinemas.py -h
usage: cinemas.py [-h] [--movies_count MOVIES_COUNT]
                  [--cinemas_count_limit CINEMAS_COUNT_LIMIT]

Script for obtaining movies list with the greatest rating                                                            
                                                                                                                     
optional arguments:                                                                                                  
  -h, --help            show this help message and exit                                                              
  --movies_count MOVIES_COUNT                                                                                        
                        Movies count for console output (10 by default)                                              
  --cinemas_count_limit CINEMAS_COUNT_LIMIT                                                                          
                        The lower bound for the cinemas count (1 by default)
```
